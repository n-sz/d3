## 概要
D3.js を使った円グラフと棒グラフのサンプルを作成します

## 使い方
1. data0からdata3にグラフで表示するvalue値となるデータを２種類入力します    
2. 決定ボタンをクリック    
3. 円グラフと棒グラフで２種類のデータを比較して表示します    
4. 円グラフはカーソルを合わせると設定値と比率が表示されます
